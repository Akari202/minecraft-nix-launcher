# Minecraft Nix Launcher

There are several other projects that do the same thing but for reasons ive forgotten i didn't want to use them. I wanted to be able to runa server with easy updadting many mods and learn some nix. Ended up being a lot of python scraping because i wanted everything to use sha256 instead of the sha512 that modrinth supplies or the sha1 that mmojang supplies. Deffinitly unneccisary. However I no longer have a single 1.7m line file with all the versions and lots of repetition so that's a bonus i guess. Feel free to use or submit a pr with fixes. In theory most versions should launch just fine but tbh i have only tested 1.21.11 at this time (i can't use java 25 on nixpkgs-24.11 lol).
